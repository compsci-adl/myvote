# MyVote

## Backend Setup

The backend is using FastAPI and uv. You can install with

```sh
uv sync
pre-commit install
```

You can run the backend with

```sh
uv run fastapi dev src/main.py
```

## Frontend setup

The frontend is using React. You can use pnpm to install and run it.

```sh
cd frontend
pnpm install
pnpm run dev
```
